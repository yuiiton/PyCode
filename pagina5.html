Controle de fluxo

No nosso cotidiano, dependemos muito de condições para realizar determinadas tarefas. Por exemplo, digamos que eu esteja no mercado e queira comprar uma bolinha de queijo, se eu tiver dinheiro suficiente na carteira, então eu vou comprar, caso contrário, eu não vou comprar.

Perceba que, para eu realizar a tarefa “comprar bolinha de queijo”, eu dependo da condição “dinheiro suficiente”, e há duas decisões aqui: uma para caso eu tenha dinheiro suficiente para comprar, e outra para caso eu não tenha dinheiro suficiente.

No mundo da programação haverá momentos em que iremos precisar verificar uma condição antes de realizar uma tarefa, e isso é chamado de estrutura de controle condicional, ou controle de fluxo.

Por exemplo, digamos que nós precisamos de um código para verificar se uma pessoa é maior de idade. Caso ela seja, a entrada dela está liberada, mas caso não seja, ela não poderá entrar. Nós poderíamos escrever esse código assim: 


idade = 20

if idade >= 18:
    print("Entrada permitida")
else:
    print("Entrada não permitida")

Vamos analisar esse código por partes. Na linha 1, a variavel idade é criada seu valor é definido como 20 (caso não entenda, veja o conteúdo de Variáveis).

Já na linha 3, temos uma verificação que se inicia com o termo if, que vem do inglês e significa se.

Em seguida, há uma condição escrita como idade >= 18, nesse trecho estamos dizendo ao computador que ele deve verificar se a variável idade possui um valor maior ou igual a 18.

Prosseguindo, temos 2 pontos, que indicam que o código abaixo só será executado se a nossa condição estabelecida for verdadeira. E se for, o computador irá imprimir a mensagem “Entrada permitida”.

A seguir nós temos o termo else, que do inglês significa se não. Ele diz ao computador o que fazer caso a nossa condição idade >= 18 não seja verdadeira. Caso não seja, ele irá imprimir na tela a mensagem “Entrada não permitida”.

Elif

Há também um outro termo na estrutura de condições, o elif. Ele serve como uma segunda condição a ser verificada caso a condição do bloco if não seja satisfeita. Vamos supor que dessa vez estamos verificando se uma pessoa precisa ou não votar. Nosso código seria algo como: 


idade = 20

if idade <= 15:
    print("Não pode votar")
elif idade >= 16 and idade < 18:
    print("Voto opcional")
else:
    print("Voto obrigatório")

Nesse código, se a idade for maior ou igual a 15, o usuário ainda não pode votar. Se for maior ou igual a 16 e menor que 18, o voto é opcional. Mas se não estiver em nenhuma dessas condições citadas, então o voto é obrigatório (estamos desconsiderando a lei que diz que pessoas com mais de 70 anos não precisam votar).

Perceba que não foi utilizado um elif idade >= 18 na última verificação da nossa estrutura, porque não há necessidade, já que se a pessoa não tiver uma idade entre 16 e 18, ela terá uma idade maior que 18.
