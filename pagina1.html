Algoritmo

Os algoritmos desempenham um papel fundamental no mundo da computação, sendo a espinha dorsal de qualquer programa ou sistema digital. Eles são sequências de instruções precisas e lógicas projetadas para resolver um problema específico ou realizar uma tarefa computacional. Desde as operações mais simples até os complexos sistemas de inteligência artificial, os algoritmos são a base sobre a qual a computação moderna é construída.

Um algoritmo pode ser comparado a uma receita de culinária: ele descreve passo a passo como realizar uma determinada tarefa. Assim como uma receita orienta um chef na preparação de um prato delicioso, um algoritmo guia um computador na execução de uma função específica.

Os algoritmos são formulados utilizando uma linguagem de alto nível que os humanos podem entender, mas que também pode ser traduzida para linguagens de programação que os computadores podem executar. Eles podem variar em complexidade, desde simples operações matemáticas até processos intricados de tomada de decisão.

Vamos supor que nós precisamos capturar quatro notas de um estudante para calcular a média de suas notas. Com algoritmos, podemos quebrar esse problema em partes menores e tratar cada uma delas individualmente. Observe o passo a passo:


1. Entrada de dados: pedimos ao usuário que insira as notas que ele deseja calcular a média, e armazenamos esses valores

2. Criação das variáveis: criamos variáveis para armazenar os valores inseridos pelo usuário, assim como também criamos uma variável para armazenar a soma dos valores que o usuário inseriu anteriormente.

3. Cálculo de média: após os passos anteriores, o computador irá pegar a soma dos valores armazenados e dividir pela quantidade de valores, para obter a média deles.

4. Saída de resultados: o computador irá exibir a média calculada para o usuário.


Linguagens de programação

Algoritmos e linguagens de programação são dois componentes interdependentes no mundo da computação. Os algoritmos, como mencionado anteriormente, são conjuntos ordenados de instruções que descrevem como resolver um problema específico. Por outro lado, as linguagens de programação fornecem as ferramentas e a sintaxe necessárias para expressar esses algoritmos de forma que os computadores possam entender e executar.

As linguagens de programação servem como um meio de tradução entre a lógica dos algoritmos e a execução real desses algoritmos em um computador. Elas oferecem uma maneira estruturada e legível para os programadores expressarem suas soluções para os problemas computacionais.

Existem muitas linguagens de programação disponíveis, cada uma com suas próprias características, sintaxe e semântica. Algumas linguagens, como C, C++, Java e Python, são amplamente utilizadas em uma variedade de aplicações devido à sua eficiência, versatilidade e suporte de comunidade. Outras linguagens, como JavaScript, são específicas para a web e são projetadas para interações com navegadores da web.

Python

Python é uma linguagem de programação de alto nível, amplamente utilizada em uma variedade de domínios, desde desenvolvimento web e científico até automação de tarefas e aprendizado de máquina. Uma das razões para a popularidade do Python é sua sintaxe simples e legível, o que o torna uma excelente escolha para iniciantes em programação.

Quando se trata de algoritmos, Python oferece uma variedade de estruturas de dados e bibliotecas que facilitam a implementação e o uso de algoritmos eficientes. Algumas características gerais da linguagem Python são:

Fácil Aprendizado: Devido à sua sintaxe simples e legível, Python é relativamente fácil de aprender, mesmo para aqueles que não têm experiência prévia em programação. Isso permite que os iniciantes façam progresso rapidamente e se sintam encorajados a continuar aprendendo.

Ampla Comunidade e Recursos de Aprendizado: Python possui uma comunidade ativa e acolhedora, com uma abundância de recursos de aprendizado disponíveis, como tutoriais, documentação oficial, fóruns de discussão e cursos online. Isso facilita para os iniciantes encontrarem suporte e recursos adicionais enquanto aprendem.

Sintaxe Simples e Legível: Python possui uma sintaxe clara e legível, o que torna o código mais fácil de entender. Isso é especialmente útil para iniciantes, pois facilita a compreensão dos conceitos de programação sem se preocupar com detalhes complexos de sintaxe.

Versatilidade: Python é uma linguagem versátil que pode ser usada em uma variedade de domínios, incluindo desenvolvimento web, análise de dados, automação de tarefas, aprendizado de máquina, entre outros. Isso permite que os iniciantes explorem diferentes áreas de interesse e descubram o que mais os motiva na programação.


Tomando como exemplo o algoritmo de cálculo de média que havíamos escrito anteriormente, aplicando-o em Python de forma bem simples seria algo semelhante a: 


nota1 = 7.5
nota2 = 8.0
nota3 = 6.5 
nota4 = 9.0 
                    
soma = nota1 + nota2 + nota3 + nota4 
media = soma / 4 
                    
print("A média das notas é:", media)

Analisando esse código por partes:

Nas linhas 1 – 4, temos uma sequência de variáveis nomeadas como nota1, nota2, nota3, nota4. Cada uma está armazenando um valor numérico referente a uma nota escolar. Note que para fazer a atribuição de um valor a uma variável, precisamos utilizar o símbolo “=” após o nome da variável, e em seguida digitar o valor que essa variável receberá.

Na linha 6 e 7, temos duas novas variáveis sendo criadas, uma nomeada “soma”, e outra nomeada “media”. Observe que, diferente das variáveis anteriores, essas duas estão armazenando equações matemáticas. A variável soma está armazenando a soma dos valores das variáveis nota1, nota2, nota3 e nota4. Já a variável media está pegando o valor da variável soma e dividindo por quatro, que é a quantidade de notas que possuímos. (OBS: lembre-se que para calcular a média de uma sequência de números, nós precisamos somar todos os números e dividir pela quantidade de números que temos, e é isso que o código está fazendo.)

Por fim, na linha 9, o código chama a função print(). Essa função em Python é utilizada para imprimir valores na tela do usuário, mostrando tudo o que está dentro dos parênteses. No caso do nosso código, o usuário receberá como resultado a mensagem: A média das notas é: 7.75
