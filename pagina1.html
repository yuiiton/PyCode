Variáveis

Uma variável é um local na memória do computador que é utilizado para armazenar um valor. Esses valores podem ser números, textos, objetos, ou qualquer outro tipo de dado (trabalharemos isso no próximo tópico). As variáveis têm um nome que as identifica e permitem que o programador acesse e manipule o valor armazenado nelas durante a execução do programa.

Temos por exemplo as seguintes variáveis: 


Nome = "Maria"
Idade = 25 
Saldo = 320.15 
Numeros_preferidos = [5,7,9,10]

Para criar o nome da variável, é preciso seguir algumas regras, algumas opcionais e outras obrigatórias, por exemplo:

1. O nome da variável deve começar com uma letra (maiúscula ou minúscula). Não é permitido começar com um número ou outro caractere especial.

2. O nome da variável pode conter apenas letras (maiúsculas ou minúsculas) e números. Outros caracteres, como espaços, pontuação ou símbolos especiais, não são permitidos.

3. Os nomes das variáveis devem ser curtos o suficiente para serem concisos, mas também devem ser longos o suficiente para serem descritivos. Evite nomes muito curtos, como "x" ou "y", que podem não transmitir informações suficientes sobre o propósito da variável. Da mesma forma, evite nomes muito longos e verbosos que dificultem a leitura do código.

4. Os nomes das variáveis devem ser descritivos e refletir o propósito ou conteúdo da variável. Isso torna o código mais legível e compreensível para outras pessoas que possam estar lendo ou mantendo o código.

Tipos de dados de variáveis

Os tipos de dados definem os diferentes tipos de valores que podem ser armazenados em uma variável. Os tipos mais comuns são:

Inteiro (int): Representa números inteiros, como 5, -10, 1000, etc.

Flutuante (float): Representa números de ponto flutuante (decimais), como 3.14, -0.5, 2.718, etc.

String (str): Representa sequências de caracteres, como "Olá, mundo!", "Python é incrível", etc.

Booleano (bool): Representa valores lógicos verdadeiro (True) ou falso (False).

Lista (list): Representa uma coleção ordenada de valores, que podem ser de tipos diferentes.

Dicionário (dict): Representa uma coleção de pares chave-valor, onde cada valor é acessado por sua chave correspondente.



Por exemplo, a nossa variável Nome possui um valor do tipo String ou str, já que ela armazena uma sequência de caracteres. Lembre-se que, para a linguagem entender uma palavra ou frase como caracteres, é preciso colocar essa palavra/frase dentro de aspas ‘’ ou aspas duplas “”.

Já a variável Idade armazena um valor do tipo Inteiro ou int.

A variável Saldo está armazenando um valor do tipo Flutuante ou float. Perceba que para definir as casas decimais nas linguagens de programação, nós utilizamos um ponto antes de escrevê-las.

E por fim, a variável Numeros_preferidos está armazenando um valor do tipo lista, já que armazena vários números em uma única variável.
