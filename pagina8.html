Funções

As funções são blocos de código que realizam uma tarefa específica e podem ser reutilizados em diferentes partes de um programa. Elas permitem organizar e estruturar o código de forma modular, facilitando a manutenção, reutilização e compreensão do código.

Vamos supor que você queira fazer um código que calcule a soma de dois valores. Com seus conhecimentos atuais, você provavelmente faria algo semelhante a isso: 


Num1 = int(input("Digite um número inteiro: ")) 
Num2 = int(input("Digite outro número inteiro: ")) 
                        
Soma = Num1 + Num2 

print(f"A soma dos números digitados é: {Soma}")

E a princípio esse código é eficaz. Mas quando trabalhamos com grandes quantidades de partes do código repetidas vezes, é ineficaz reescrever elas ao longo do código, e é pra isso que existem as funções. Com elas, é possível reutilizar trechos do seu código em outras partes dele, sem precisar reescrevê-lo.

Estrutura de uma função

Para definir uma função em Python, usamos a palavra-chave def, seguida pelo nome da função e, opcionalmente, parâmetros entre parênteses. A estrutura geral seria algo similar a: 


def nome_da_funcao (parâmetros):
    #codigo aqui

Parâmetros de uma função

Os parâmetros de uma função são utilizados quando essa função precisa receber valores de outras partes do código para tratar desses valores dentro da função. Por exemplo, utilizando o código escrito anteriormente, poderíamos fazer uma função assim: 


def soma(a,b):
    return a + b

Nessa função, os parâmetros são as variáveis a e b, que serão utilizados dentro da função para realizar uma operação matemática. E então, a função vai retornar a soma dos valores a e b. (return a + b)

Escrevendo o código inteiro, ficaria assim: 


def soma (a, b): 
    return a + b 
                            
Num1 = int(input("Digite um número inteiro: ")) 
Num2 = int(input("Digite outro número inteiro: ")) 
                        
print(f"A soma dos valores é {soma(Num1, Num2)}")

Para chamar a função, nós escrevemos seu nome e dentro dos parênteses colocamos os parâmetros, se houver. Note que nós atribuímos os valores de Num1 e Num2 aos valores de a e b, para a função realizar os cálculos.

Existe uma ampla quantidade de assuntos que envolvem as funções, e o que foi mencionado acima é só o básico. Por isso, continue pesquisando para aprender cada vez mais como e quando utilizar as funções.
