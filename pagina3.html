Operadores matemáticos

Os operadores matemáticos são símbolos especiais que são utilizados para realizar operações aritméticas em números nas linguagens de programação. Os principais operadores matemáticos em Python são:

1. Adição (+): Usado para somar dois ou mais números.

Subtração (-): Usado para subtrair um número de outro:

Multiplicação (*): Usado para multiplicar dois ou mais números.

Divisão (/): Usado para dividir um número pelo outro.

Exponenciação (**): Usado para elevar um número à potência de outro.

Há muitos outros operadores matemáticos, mas no geral, esses são os principais que você precisa saber agora. Portanto, não esqueça de continuar pesquisando mais a fundo sobre esses conteúdos se sentir necessidade.
